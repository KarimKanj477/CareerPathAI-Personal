# Technology Stack

## Overview

CareerPath AI will be developed using modern web technologies. Each technology has been selected according to its role in the application.



## Frontend

- React
- TypeScript
- HTML5
- CSS3
- Bootstrap

The frontend will provide the user interface and communicate with the backend through REST APIs.



## Backend

- Java
- Spring Boot

The backend will handle business logic, authentication, career management, roadmap generation, and communication with the database.



## Database

- MySQL

MySQL will store all application data, including users, careers, skills, learning resources, and personalized roadmaps.



## AI Services

- Google Gemini API

Gemini will be used to generate AI-powered career recommendations and answer career-related questions.



## Development Tools

- Git
- GitHub
- Jira
- Postman
- Docker
- Visual Studio Code

These tools will be used for version control, project management, API testing, containerization, and software development.



## Why These Technologies?

These technologies are widely used in modern software development. They are reliable, well documented, and suitable for building scalable web applications. Using this stack will also help the development team collaborate efficiently throughout the project.
